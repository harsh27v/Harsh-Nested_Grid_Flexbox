## Dr.Nut — Responsive Hero Section (Assignment 4)
Welcome to the Dr.Nut Soda Hero Section project!

## 💡 Purpose
This project was created for Assignment 4 – Nested Grid & Flexbox Layout.
The project demonstrates:
- CSS Grid layout using a custom grid.css grid system
- Nested Grid structure for organizing hero content
- Flexbox for button alignment
- Responsive images using the <picture> element
- Order property to change layout order on mobile
- Mobile-first responsive design

## Installation 💻
No installation required, simply download the zip file and extract.

## Usage
Open index.html in your choice browser.
